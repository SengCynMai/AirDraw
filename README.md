# FaceVerse

## Project Introduction

FaceVerse is a real-time face filter project using Python and OpenCV. The system detects faces through a webcam and adds filters such as sunglasses, hats, and masks in real time.

This project uses computer vision techniques like face detection, facial landmark tracking, and image overlay processing. The program runs directly in Visual Studio Code using webcam input.

---

## How the Project Will Be Developed

The project will be developed using Python, OpenCV, and MediaPipe. The webcam will capture video, and the system will detect facial landmarks such as eyes and nose positions. Then, filter images will be placed on the face correctly.

Development steps:

1. Capture webcam video using OpenCV  
2. Detect face and facial landmarks  
3. Apply filters in real time  
4. Add multiple filters and keyboard controls  
5. Test and improve the system

## Why I Chose This Project

I chose this project because it is fun and interesting. Face filters are popular in social media, and I wanted to learn how they work using computer vision.

This project also helps me practice important OpenCV skills such as face detection, image processing, and real-time video processing.
